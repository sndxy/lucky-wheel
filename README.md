# 🎡 專業級線上幸運大轉盤 (聲效與批次匯入加強版)

這是一個基於 HTML5 Canvas 與 Web Audio API 開發的高級抽獎工具。不僅擁有流暢的物理動畫，更解決了瀏覽器對音效播放的限制，並支援大規模名單匯入。

## ✨ 核心特色
- **🎶 智慧聲效系統**：採用 Web Audio API 即時生成電子音訊，徹底解決外部 MP3 連結失效與瀏覽器自動播放限制問題。
- **📥 大規模批次匯入**：支援從 Excel 複製貼上人名，或直接讀取 `.txt`、`.csv` 檔案，一秒處理百人名單。
- **🎯 自然擺動指針**：指針具備物理碰撞模擬，經過格線時會產生回彈動畫與滴答聲。
- **🎨 三大主題切換**：
    - **🌿 莫蘭迪**：簡約清新，適合專業會議。
    - **🌌 賽博深邃**：酷炫高對比，適合晚會活動。
    - **☕ 暖陽焦糖**：溫馨舒適，適合私人聚會。
- **⚖️ 獲獎排除機制**：點擊「排除參加」即可將中獎者移出轉盤，名單色彩保持不變，維持視覺美觀。
- **📱 響應式設計**：完美適配手機與平板電腦，支援觸控操作。

## 🚀 快速上手
1. **單筆新增**：在右側輸入框輸入姓名並點擊「加入」。
2. **批次匯入**：
   - 切換至「批次匯入」標籤。
   - 直接貼上由逗號或換行隔開的名單。
   - 或點擊「讀取檔案」選取您的名單檔案。
3. **設定獎項**：自訂本次抽獎的獎品名稱。
4. **開始抽獎**：點擊「立即開始抽獎」，享受聲畫同步的緊張感。

## 🛠️ 技術規格
- **Frontend**: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
- **Graphics**: HTML5 Canvas API
- **Audio**: Web Audio API (OscillatorNode)
- **File I/O**: FileReader API API

## 🔒 安全性說明
- **純靜態網頁**：本工具不包含後端資料庫，所有資料僅存在於您的瀏覽器中，重新整理即清空，保護隱私不外洩。
- **HTTPS 支援**：建議部署於 GitHub Pages，享受加密傳輸保護。
