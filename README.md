# 🎡 精美互動幸運大轉盤

這是一個基於 HTML5 Canvas 開發的網頁抽獎工具，具備流暢的物理動畫與美觀的介面設計。

### ✨ 核心特色
- **物理模擬指針**：指針會隨著轉盤格線產生自然的彈撥擺動效果。
- **三種風格切換**：內建「🌿 莫蘭迪淺色」、「🌌 賽博深邃」與「☕ 暖陽焦糖」三種主題。
- **靈活名單管理**：支援即時新增/刪除參加者，並自動計算轉盤比例。
- **獲獎排除機制**：中獎後可選擇將該成員排除於轉盤外，同時保留中獎歷史紀錄。
- **全中文化介面**：親切的繁體中文操作界面，適合各種聚會與活動。

### 🚀 如何使用
1. 在左側「參加成員」輸入名字並點擊加入。
2. 設定本次抽獎的「獎項名稱」。
3. 點擊「立即開始抽獎」等待結果。
4. 中獎後，可點擊「排除參加」確保該人不會重複中獎。

### 🛠️ 技術使用
- HTML5 / CSS3 (Flexbox & Grid)
- Vanilla JavaScript (無須外部套件)
- Canvas API
